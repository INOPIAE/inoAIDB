Ziel von inoAIDB ist es, Interessierten die Möglichkeit zu geben, schnell herauszufinden, welche Applikationen KI nutzen.

Fehlt eine Anwendung, so kann diese über ein Forms-Formular https://forms.office.com/e/s1RXAkWSV3 mitgeteilt werden. Es wird versucht, diese Daten zeitnah einzugeben.

Die Übersicht der Anwendung kann auch per API abgerufen werden (siehe API Dokumentation).
